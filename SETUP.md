# ⚙️ Setup Guide – FoodieHub

## Prerequisites
- Java 8 or higher
- Eclipse IDE
- Apache Tomcat
- MySQL Server

## Steps to Run
1. Clone the repository
2. Import project into Eclipse as a Dynamic Web Project
3. Configure Apache Tomcat in Eclipse
4. Execute SQL scripts from Database Structure folder
5. Update database credentials in DBConnection.java
6. Run the project on Tomcat
7. Access the application via browser

## Notes
- Sample data is provided for demo purposes
- Database credentials should not be committed
